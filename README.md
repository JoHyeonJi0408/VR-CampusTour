### 🗂️ 프로젝트 소개  
>4인 졸업 프로젝트입니다.  
졸업 전시회에서 눈에 띄는 작품을 선보이고, 새로운 기술을 연구해보고자 VR 캠퍼스 투어를 기획했습니다.  
OVR를 활용하여 VR을 구현하고 Unity를 사용하여 개발하였습니다.
